# AQI & Weather Analysis (India)

## Objective
Analyze how weather variables influence air pollution (PM2.5, PM10, NO2, etc.) across Indian cities.

## Dataset
- Source: Kaggle
- Cities: Multiple Indian cities
- Variables: AQI components, weather, wind, UV

## Methodology
- Data cleaning & validation
- Exploratory Data Analysis
- Correlation & city-level analysis

## Key Insights
- Correlation Heatmap

- PM2.5 shows moderate correlation with NO2 and PM10
- Weather variables have weaker but city-dependent influence
